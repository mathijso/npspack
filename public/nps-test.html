NPS Widget Test

Deze pagina laadt het NPS Widget script.

De widget zou na een paar seconden moeten verschijnen (als je niet recentelijk hebt ingevuld via deze browser).

Instructies:

  1. Zorg dat je Laravel development server draait (php artisan serve).
  2. Vervang JOUW_GELDIGE_SITE_ID in de script tag hieronder met een public_id van een site die je hebt aangemaakt in de applicatie (via de /sites pagina -> Script knop).
  3. Open dit bestand direct in je browser (via file://... pad of serveer het via een lokale webserver).
